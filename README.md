## 教程

使用主题默认您已创建过[hexo](https://hexo.io)项目

### 使用

将本仓库的代码复制到您的hexo项目:

```bash
$ git clone https://github.com/can-dy-jack/hexo-theme-delicate themes/delicate
```

或者可以下载本仓库的代码，解压到theme文件夹之下。

之后只需更改配置 `_config.yml`：

```yml
theme: delicate
```

### 本主题自定义 front-matter

1. 置顶文章

```yml
top: true
```
